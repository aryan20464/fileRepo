2.4.3. Unique X.509 Mapping should be present
Control Statement
X.509 specifies standard formats for certificates when the default Identity Asserter is used in WebLogic Server.
The attribute value is a wayof distinguishinguser accounts from one another.
Risk / Impact
If attribute values are not unique, a less privileged user could inadvertently be granted the access rights of a
more trusted userwith the same attribute value
Risk Rating
Medium
Implementation Steps
It should be ensured that the attribute value used to map from X.509 certificates to user accounts is unique
within the certificate authority'sdomain ofusers.
